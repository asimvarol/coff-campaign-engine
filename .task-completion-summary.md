### 3. Brand DNA API Routes (Next.js)
**Location**: `apps/web/src/app/api/brands/`

**Endpoints**:
- `GET /api/brands` - List all brands
- `GET /api/brands/:id` - Get single brand
- `POST /api/brands/analyze` - Analyze URL and create brand
- `PUT /api/brands/:id` - Update brand
- `DELETE /api/brands/:id` - Delete brand
- `POST /api/brands/:id/analyze` - Re-analyze brand

**Data Store**: In-memory `brandStore` with HMR persistence (`data-store.ts`)

**Features**:
- Mock brand analyzer with deterministic colors
- Pre-populated demo brands (Golden Horn Jewellery, Urban Fitness Co)
- Full CRUD operations
- TypeScript type safety with `MockBrandDNA` interface
